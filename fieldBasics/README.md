# Field Basics

This folder contains fundamental implementations for working with finite fields, prime numbers, cyclic groups, and cryptographic hash functions. These are essential building blocks for many cryptographic algorithms.

## Overview

The `fieldBasics` module provides utilities for:
- **Prime number operations**: Checking primality and finding primes
- **Number factorization**: Prime factorization and related operations
- **Modular arithmetic**: Fast modular exponentiation
- **Generators and primitive roots**: Finding generators of multiplicative groups
- **Cyclic groups**: Analyzing cyclic subgroups of finite fields
- **Hash functions**: Cryptographic hash utilities

## Files

### `prime.py`
Prime number utilities for cryptographic applications.

**Functions:**
- `is_prime(n)`: Check if a number is prime
- `find_prime_around(target)`: Find a prime number near a target value

**Example:**
```python
from prime import is_prime, find_prime_around

print(is_prime(97))  # True
prime = find_prime_around(10**9)  # Find prime near 1 billion
```

### `generator.py`
Core functions for factorization, modular arithmetic, and finding generators (primitive roots) of multiplicative groups modulo a prime.

**Functions:**
- `factorize(n)`: Factorize a number into prime factors with powers
- `get_prime_factors(n)`: Get only the prime factors (without powers)
- `format_factorization(n)`: Get factorization as a formatted string
- `modular_power(base, exponent, modulus)`: Fast modular exponentiation
- `find_generators(P)`: Find all generators (primitive roots) modulo prime P
- `divide_prime_by_factors_of_p_minus_one(P)`: Calculate (P-1)/q for each prime factor q of (P-1)

**Example:**
```python
from generator import factorize, find_generators, modular_power

factors = factorize(12)  # {2: 2, 3: 1}
result = modular_power(3, 5, 7)  # 3^5 mod 7 = 5
generators = find_generators(7)  # [3, 5]
```

### `cyclic_group.py`
Functions for analyzing cyclic groups and their subgroups in finite fields.

**Functions:**
- `get_cyclic_group_elements(g, P, order=None)`: Get all elements of a cyclic group generated by g modulo P
- `get_cyclic_groups(P)`: Analyze all cyclic subgroups of the multiplicative group modulo prime P

**Example:**
```python
from cyclic_group import get_cyclic_groups

result = get_cyclic_groups(23)
# Returns detailed information about:
# - Factorization of P-1
# - All primitive roots
# - Cyclic subgroups for each prime factor
```

### `hash.py`
Hash function utilities for cryptographic applications.

**Functions:**
- `hash_string(data, algorithm='sha256')`: Hash a string using MD5, SHA1, SHA256, or SHA512
- `simple_hash(data)`: Simple hash using Python's built-in hash function

**Example:**
```python
from hash import hash_string

hash_value = hash_string("hello world", 'sha256')
```

## Usage

### Basic Prime Field Operations

```python
from prime import is_prime, find_prime_around
from generator import find_generators, modular_power

# Find a prime
P = find_prime_around(1000)
print(f"Prime: {P}")

# Find generators
generators = find_generators(P)
print(f"Generators: {generators}")

# Modular exponentiation
result = modular_power(generators[0], 10, P)
print(f"g^10 mod P = {result}")
```

### Analyzing Cyclic Groups

```python
from cyclic_group import get_cyclic_groups

P = 23
result = get_cyclic_groups(P)

print(f"Prime: {result['prime']}")
print(f"P-1 = {result['p_minus_one']}")
print(f"Factorization: {result['factorization_str']}")
print(f"Primitive roots: {result['all_primitive_roots']}")

# Access cyclic subgroups
for q in result['prime_factors']:
    subgroup = result['cyclic_subgroups'][q]
    print(f"Subgroup for factor {q}:")
    print(f"  Generator: {subgroup['generator']}")
    print(f"  Order: {subgroup['order']}")
    print(f"  Elements: {subgroup['elements']}")
```

## Mathematical Background

### Finite Fields
A finite field (Galois field) of prime order p, denoted GF(p) or F_p, consists of the integers {0, 1, 2, ..., p-1} with addition and multiplication modulo p.

### Multiplicative Group
The multiplicative group modulo p, denoted (Z/pZ)* or F_p*, consists of {1, 2, ..., p-1} under multiplication modulo p. This group is cyclic of order p-1.

### Generators (Primitive Roots)
A generator g of (Z/pZ)* is an element such that every element of the group can be written as g^k for some k. The number of generators is φ(p-1), where φ is Euler's totient function.

### Cyclic Subgroups
For each prime factor q of p-1, there exists a cyclic subgroup of order q^k (where k is the power of q in the factorization of p-1). These subgroups are generated by g^((p-1)/q) where g is a primitive root.
